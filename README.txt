PROJECT INFORMATION
===================

1. Project id is configured in the property file
2. Scheduler can be configured to run on defined time
3. MonoDB is used to store the information
4. Web service end point to view the result is "/hometest/weeklySummary" (Ex: http://localhost:8080/hometest/weeklySummary)
5. Swagger UI url is "<host>/swagger-ui.html" (Ex: http://localhost:8080/swagger-ui.html)
6. Maven command to build the project is "mvn clean install"
7. Maven command to run the application is "java -jar -jar target/hometest-0.0.1-SNAPSHOT.jar"
8. Maven command to run the application in debug mode is "java -Xdebug -Xrunjdwp:server=y,transport=dt_socket,address=8000,suspend=n -jar target/hometest-0.0.1-SNAPSHOT.jar"
